# DERConnect Testbed Projects
This repository contains code, presentations, and documentation from work I've completed while at DERConnect. Summaries listed below...

# Custom Building Automation Software
Assisted in testing of smart devices and automation programs to aid in the development of OCCTOPI, a custom building automation and optimization software. The project aimed to see at least a 20% reduction in plug load and lighting energy consumption. The long-term purpose is to reduce the building's electrical load so that other systems can be electrified. Further development of the project is ongoing at DERConnect. 

# VAV Optimization
By utilizing Variable Air Volume (VAV) devices already controlled by UCSD's campus wide building automation software, this project sought to utilize motion and temperature sensors, along with pre scheduled work hour ranges, to reduce time spent in occupied mode for VAV devices in academic buildings. These reductions turned VAV devices to unoccupied mode when office spaces were unoccipied, while also utilizing time averaging on/ off periods to reduce the electrical load required by building air handlers. This project was able to reduce simulated occupatied mode time by 51.3% for a large office space in Center Hall.

# Microgrid Project-in-a-box
DERConnect conducts various outreach programs and events with community members to showcase lab work, perform informative seminars, and increase community involvement and engagement with the labs goals and projects. The project in a box series sought to educate community members on the foundational engineerings principles of the lab's work. The microgrid optimization project created an interactive program where users could build their own microgrid to maximize cost savings from buying electricity from outside their microgrid. They were able to analyze graphical outputs of demand and generation from renewable energy generators in their grid setup to find the best working conditions. 

# Inverter Testbed
A collaboration project with Critical Loop, an electrical power solutions company based out of Long Beach, this project seeks to build an Inverter Testbed within the wider DERConnect Testbed to analyze inverter performances and capabilities within rapidly varying grid demand conditions. Project is ongoing.
